📊 BMW Global Sales Performance Dashboard (2018–2025)

Project Overview:
This project analyzes BMW's global sales data using Power BI to evaluate revenue trends, electric vehicle (BEV) growth, regional performance, and macroeconomic impact.

📌 Objectives:
- Analyze revenue growth trends
- Study EV (BEV) market expansion
- Evaluate regional sales performance
- Understand impact of GDP growth & fuel price index

📈 Key Insights:
- Revenue shows consistent multi-year growth
- BEV share is steadily increasing
- Premium segment remains strong
- Economic indicators influence automotive sales performance

Tools Used:
- Power BI
- DAX
- Data Modeling
- Business Intelligence
